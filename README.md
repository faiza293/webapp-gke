# webapp-gke

# Project structure 

Backend Application (newapp-backend)
webapp-gke/
└── backend/
    ├── app.py
    ├── backend-deployment.yaml
    ├── backend-service.yaml
    ├── Dockerfile
    ├── requirements.txt
    └── venv/  ( virtual environment folder. This is optional)

Frontend Application (react-frontend)
webapp-gke/
└── react-frontend/
    └── frontend/
        ├── Dockerfile
        ├── node_modules/  (Generated by npm install)
        ├── package.json
        ├── package-lock.json
        ├── public/
        │   ├── index.html
        │   └── ... (other public assets)
        ├── README.md
        └── src/
            ├── App.js
            ├── App.css
            ├── index.js
            ├── index.css
            └── ...
